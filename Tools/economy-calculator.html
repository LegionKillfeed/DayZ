Base Radar Commands

    /base radar add тоз
    • Create a base radar of maximum 999metres
    • X Coord: XXXX (First coord in coords from Logs or iZurvive, not GPS)
    • Z Coord: ZZZZ (Second coord in coords from Logs or iZurvive, not GPS)
    • Radius: 999metres Maximum
    • Zone Name: Whatever you want, usually faction names
    • Zone Owner: Whoever is in charge of radar (They can self manage base ignore list), if it's for the server not a player/faction, I usually set myself
    • Zone Channel: Channel to assign it to or leave empty to make one
    • Zone Role: Role for people to have access to the channel (Faction role usually) or leave empty to make one
    • Ban Build: Auto-ban on build
    • Ban Kill: Auto-ban on kill
    • Ban Death: Auto-ban on death
    • Ban Entry: Auto-ban on entry
    • Ban Disconnect: Auto-ban on logout/disconnect
    • Base Location: Turn locations on/off in radar channel
    • Base Ping: Ping faction role/channel role on base radar notifications
    • (Admin/Owner/Founder)
    /base radar edit тоз
    • Edit an existing base radar.
    • Zone ID: Number from footer of base radars, no letters.
    • X Coord: XXXX (First coord in coords from Logs or iZurvive, not GPS)
    • Z Coord: ZZZZ (Second coord in coords from Logs or iZurvive, not GPS)
    • Radius: 999metres Maximum.
    • Zone Name: Whatever you want, usually faction names.
    • Zone Channel: Add to create, edit to assign existing
    • Existing Channel: Existing channel to assign if not assigning, leave blank.
    • Zone Owner: Whoever is in charge of radar (They can self manage base ignore list), if it's for the server not a player/faction, I usually set myself.
    • Zone Role: Role for people to have access to the channel (Faction role usually) or leave empty to make one.
    • Ban Build: Auto-ban on build.
    • Ban Kill: Auto-ban on kill.
    • Ban Death: Auto-ban on death.
    • Ban Entry: Auto-ban on entry.
    • Ban Disconnect: Auto-ban on logout/disconnect.
    • Base Location: Turn locations on/off in radar channel.
    • Base Ping: Ping faction role/channel role on base radar notifications.
    • (Admin/Owner/Founder)
    /base radar delete тоз
    • Delete a base radar
    • Zone ID: Number from footer of base radars, no letters.
    • (Admin/Owner/Founder)
    /base radar list тоз
    • Get list of a base radars.
    • (Admin/Owner/Founder)
    /base ignore add тоз
    • Add gamertag/gamertags to base radar ignore.
    • Must be base owner (or Moderator/Admin/Owner/Founder)
    • Must be used in radar channel - will not work outside channel.
    /base ignore remove тоз
    • Remove gamertag/gamertags from base radar ignore
    • Must be base owner (or Moderator/Admin/Owner/Founder)
    • Must be used in radar channel - will not work outside channel.
    /base ignore info тоз
    • Lists base radar details
    • Lists base ignore list too
    • Must be base owner (or Moderator/Admin/Owner/Founder)
    • Must be used in radar channel - will not work outside channel.

Experimental Commands

    /types server boost тоз
    • Boost your types file by 'x' amount
    • (Owner/Founder)
    /types local edit тоз
    • Bulk edit local types file by 'category', 'tag', 'usage' or 'value'
    • (Owner/Founder)
    /types local validate тоз
    • Upload a local types file to validate
    • (Owner/Founder)
    /admin spawner тоз
    • Build an event from discord
    • Selection: Items/Static/Vehicle
    • X Coord: XXXX (First coord in coords from Logs or iZurvive, not GPS)
    • Z Coord: ZZZZ (Second coord in coords from Logs or iZurvive, not GPS)
    • Type Name: Item/Object/Vehicle name from types.xml
    • Lifetime: How long the item stays spawned for in seconds (permanent spawn set to 3888000)
    • Player Name: Name to reference event in files (whatever you want, but no spaces)
    • Nominal: How many to spawn at location
    • Max Amount: Maximum items to spawn at location
    • Min Amount: Minimum items to spawn at location
    • Restock: Seconds to wait before respawning item (usually 0)
    • (Owner/Admin/Founder)

Player Commands

    • Amount: 100 or 100.00 or 101.01 (but no extra symbols or letters).
