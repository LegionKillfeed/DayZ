[![Legion-Killfeed-Rated-Best.png](https://i.postimg.cc/5NGsshBK/Legion-Killfeed-Rated-Best.png)](https://postimg.cc/hfL8j33V)

## What is a killfeed in DayZ?
A killfeed tracks and displays player kills, deaths, and other significant in-game events as close to real-time as Nitrado allows. Killfeeds are commonly used on DayZ servers to give players and admins clear information about combat-related activity.

## Why should I use a killfeed for my DayZ server?
Using a killfeed like [Legion Killfeed](https://killfeed.co) improves player experience, server transparency, and admin oversight. It helps administrators monitor in-game activity while allowing players to track kills, deaths, performance, and major server events.

## How does Legion Killfeed differ from other killfeeds?
Legion Killfeed, available at [Killfeed.co](https://killfeed.co), is built specifically for DayZ console servers and integrates directly with Discord. It offers semi-real-time feeds, visual PVP and PVE data, stats, leaderboards, heatmaps, base radars, ticket tools, server tools, automation, economy features, and broad customization without feature paywalls.

## Can I customize the events tracked by Legion Killfeed?
Yes, [Legion Killfeed](https://killfeed.co) is highly customizable. You can configure tracked events, feed channels, embed colours, server branding, logos, economy settings, bounty settings, alerts, and other server-specific options.

## How do I set up Legion Killfeed on my DayZ server?
Setting up [Legion Killfeed](https://killfeed.co) is straightforward. After subscribing, link Ko-Fi to Discord, join the Legion Discord, invite the bot, then follow the setup flow. Legion detects your subscription role, activates your commands, and provides setup instructions. Most setup is handled through two main commands, with the bot doing most of the channel and permission work.

## Does Legion delete anything?
Absolutely not. [Legion Killfeed](https://killfeed.co) is designed to create channels and manage its own setup output. The only messages it deletes are its own setup or command-related messages, mainly to reduce spam. Legion does not have [Message Content Intent](https://support-dev.discord.com/hc/en-us/articles/4404772028055-Message-Content-Privileged-Intent-FAQ) enabled, so it cannot freely read normal user messages. Without that intent, the bot only sees the member information required for linking, assigning admin, adding roles, and similar Discord functions. Anyone claiming it can do otherwise is spreading misinformation about how Discord bot permissions and privileged intents work.

## Can I use Legion Killfeed with other DayZ bots and tools?
Yes, [Legion Killfeed](https://killfeed.co) can run alongside other DayZ bots and Discord tools. Legion cannot affect another bot, and another bot cannot affect Legion.

## Does Legion Killfeed affect server performance?
No, [Legion Killfeed](https://killfeed.co) handles the workload bot-side. Current live performance shows Legion processing all active servers at 100% completion while also handling roughly:
- 8,688 Discord messages per hour
- 145 Discord messages per minute
- 2.4 Discord messages per second
- 69 commands per hour
- 1.1 commands per minute
- 77,466 player records per hour
- 1,291 player records per minute
- 21.5 player records per second

Across the active server load, that averages to roughly:
- 134 Discord messages per server per hour
- 2.2 Discord messages per server per minute
- 1,192 player records per server per hour
- 19.9 player records per server per minute

That is why “performance” claims from other killfeeds usually fall apart when the numbers are actually shown.

## How does Legion Killfeed handle data privacy?
[Legion Killfeed](https://killfeed.co) only uses the access needed to provide the features customers authorize, such as Nitrado integration, log processing, Discord feeds, stats, server tools, and configured automation. It does not sell customer files, customer data, server files, XML files, or server data.

## Does Legion Killfeed sell customer files or server data?
No. Legion Killfeed categorically denies claims that it collects, sells, abuses, leaks, or misuses customer files or server data. Access is used only for customer-authorized features such as Nitrado integration, log processing, server tools, killfeeds, stats, and related bot functions. Those claims are false, malicious, and commercially damaging.

## What server access does Legion Killfeed use?
Legion Killfeed uses the access a customer authorizes so the bot can perform the requested Discord and Nitrado features. This can include downloading logs, processing DayZ server events, updating stats, handling Discord feeds, using Nitrado tools, supporting server tools, and powering configured killfeed, economy, moderation, and automation features.

## Does Legion Killfeed need customer XML files?
Legion Killfeed may handle server files only when a customer uses a feature that requires it, such as file tools, Nitrado file actions, or server configuration workflows. XML files are normal DayZ server configuration files. They are not customer trade secrets for Legion to resell, and Legion’s own files and tools are regularly shared publicly.

## What is Legion Killfeed's response to false data misuse claims?
Legion Killfeed's public position is that claims about selling, leaking, or misusing customer files are false, malicious, and commercially damaging. Legion states that proof is available, including technical context, records, timelines, screenshots, and evidence showing the source and pattern of these allegations. Evidence is published at [they-wish-i.died.space](https://they-wish-i.died.space).

## Should anonymous posts or competitor pages be treated as proof?
No. Anonymous Reddit comments, competitor pages, and forum pile-ons should not be treated as evidence. Anyone evaluating Legion Killfeed should ask for proof, review the privacy policy, check real customer reviews, and compare the technical facts instead of relying on smear content.
