Privacy Policy

    Welcome to Legion Killfeed! This privacy policy explains how we handle your data when you use our bot to download logs from Nitrado.net using API tokens. By using our bot, you agree to the terms outlined in this policy.

Data Collection

    We collect the following data:

  • API Tokens: We collect Nitrado API tokens to access and download logs from your Nitrado.net account.
  • Discord Data: We collect and store the following data from Discord:
    • Server ID
    • Channel ID
    • User IDs for linked accounts
    • Messages related to log outputs
  • Logs Data: We collect and store the data from logs as stats

Data Usage

    We use your data as follows:

  • API Tokens: Tokens are used solely to authenticate and access logs from Nitrado.net.
  • Discord Data: Discord Data is used to deliver and display logs in your Discord server.
  • Logs Data: Server Data is used to retrieve logs and update stats.
  • Server Stats: Server Logs are used to update stats for your servers using Legion Killfeed.
  • Global Stats: Server Logs are used to update stats globally across all servers using Legion Killfeed.

Data Storage

    We store your data as follows:

  • API Tokens: Tokens are securely stored and used only by the bot unless express written consent is given to use it for bot testing.
  • Discord Data: Data is stored temporarily to ensure the proper functioning of the bot.
  • Player Stats: Server Specific Player Stats are stored up to 12 months after bot removal before being completely wiped from the database.
  • Global Stats: Global Player Stats are stored permanently, no personal information is retained, only gamertags.

Data Retention and Deletion

    Upon bot removal:

  • All related Discord data, including channel and user information, will be automatically deleted from our database.
  • The Nitrado API token associated with your account will also be deleted immediately.

Security

    We prioritize the security of your data and implement industry-standard practices to protect it. However, please note that no method of transmission over the internet or electronic storage is 100% secure.
